# 策略模式 vs 工廠模式

- 工廠模式: 訓練營 訓練出不同 冒險者 去進行 任務 (關注 物件如何被生產)
- 策略模式: 冒險者 使用不同的 策略　 去解決 任務 (關注 行為如何被封裝)
